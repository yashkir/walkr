# walkR

A place for creating, sharing, and viewing guided walks.

Includes a forum to discuss walks.

## Try it

[Heroku](https://randy-walkr.herokuapp.com/)

## Features

- Create guided walks
- View other's guided walks
- Comment on walks
- Talk about various topics in the forums

## Technologies

- Python
- Django
- Gunicorn
- Bootstrap4
- S3
